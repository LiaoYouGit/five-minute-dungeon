# 粒子系统增强指南

## 新增功能总览

### 1. 粒子形状 (Shape)
- **rect**: 矩形粒子（默认）
- **circle**: 圆形粒子（更自然）

### 2. 发光效果 (Glow)
- **glow**: boolean - 是否启用发光
- **glowIntensity**: number - 发光强度（1-3）

### 3. 物理效果
- **gravity**: number - 重力（像素/秒）
- **wind**: number - 风力（像素/秒）
- **drag**: number - 速度衰减系数（默认0.98）
- **bounce**: boolean - 碰撞反弹
- **bounceDecay**: number - 反弹衰减（默认0.7）

### 4. 旋转动画
- **rotationSpeed**: number - 旋转速度（弧度/秒）

### 5. 缩放动画
- **scaleAnimation**: boolean - 是否启用缩放动画
- **scaleStart**: number - 初始缩放（默认1）
- **scaleEnd**: number - 结束缩放（默认0）

### 6. 拖尾效果
- **trail**: boolean - 是否启用拖尾
- **trailLength**: number - 拖尾长度（默认5）

### 7. 方向发射
- **direction**: { angle: number, spread: number } - 方向发射（角度+扩散）

## API使用示例

### 基础用法（兼容旧API）
```javascript
// 默认矩形粒子，无增强效果
this.particles.emit(x, y, 10, {
  colors: ['#ff6b35', '#ffa500'],
  speed: 80,
  life: 0.5
});
```

### 圆形粒子 + 发光（火焰特效）
```javascript
this.particles.emit(x, y, 20, {
  colors: ['#ff5522', '#ffaa00', '#fff'],
  speed: 150,
  life: 0.7,
  shape: 'circle',          // 圆形粒子
  glow: true,               // 发光效果
  glowIntensity: 2,         // 强发光
  gravity: 80,              // 向下重力
  scaleAnimation: true,     // 缩放动画
  scaleStart: 1.5,          // 初始大粒子
  scaleEnd: 0.2             // 结束小粒子
});
```

### 拖尾粒子（轨迹效果）
```javascript
this.particles.emit(x, y, 30, {
  colors: ['#4ecdc4', '#fff'],
  speed: 150,
  life: 1.2,
  shape: 'circle',
  glow: true,
  trail: true,              // 拖尾效果
  trailLength: 8,           // 8帧拖尾
  gravity: 30               // 微弱重力
});
```

### 旋转粒子（旋转特效）
```javascript
this.particles.emit(x, y, 20, {
  colors: ['#ffcc00', '#fff'],
  speed: 100,
  shape: 'rect',            // 矩形粒子（旋转效果明显）
  rotationSpeed: 5,         // 快速旋转
  gravity: 50
});
```

### 碰撞反弹（地面粒子）
```javascript
this.particles.emit(x, y, 15, {
  colors: ['#fff', '#ffcc00'],
  speed: 200,
  gravity: 150,             // 强重力
  bounce: true,             // 碰撞反弹
  bounceDecay: 0.6,         // 反弹衰减
  shape: 'circle'
});
```

### 方向发射（定向特效）
```javascript
// 向上发射（角度: -π/2，扩散: 30度）
this.particles.emit(x, y, 10, {
  colors: ['#ff0000'],
  speed: 200,
  direction: {
    angle: -Math.PI / 2,    // 向上（-90度）
    spread: 30              // 扩散30度
  },
  gravity: 100              // 重力回落
});
```

## 性能建议

### 粒子数量控制
- **战斗特效**: ≤20粒子（高频触发）
- **Boss特效**: ≤50粒子（低频触发）
- **升级特效**: ≤30粒子（低频触发）

### 对象池大小
默认200粒子池，可根据实际需要调整：
```javascript
const particles = new ParticleSystem(300); // 扩大到300
```

### 世界边界设置
启用bounce前需设置世界边界：
```javascript
particles.setWorldBounds({
  minX: 0,
  maxX: mapWidth,
  minY: 0,
  maxY: mapHeight
});
```

## 渲染顺序

粒子系统采用分层渲染：
1. **拖尾层**（先渲染，在粒子后面）
2. **粒子层**（后渲染，覆盖拖尾）

确保拖尾效果看起来自然（粒子在前，轨迹在后）。

## 最佳实践

1. **圆形粒子更适合发光效果**（shadowBlur与圆形配合更好）
2. **矩形粒子适合旋转效果**（旋转动画更明显）
3. **发光强度控制在1-3**（过高会影响性能）
4. **拖尾长度控制在3-10**（过长会降低性能）
5. **重力效果与速度匹配**（速度越高，重力可以更大）
6. **高频特效粒子数≤20**（避免性能问题）
