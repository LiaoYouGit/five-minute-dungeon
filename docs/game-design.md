# 五分钟地牢 — 游戏设计文档

## 1. 游戏概述

**名称：** 五分钟地牢 (5-Minute Dungeon Rush)
**品类：** Roguelite / 超休闲 / 幸存者类
**平台：** 移动端（竖屏）优先，PC 浏览器兼容
**单局时长：** 严格 5 分钟
**操作方式：** 单手虚拟摇杆（移动端）/ WASD（PC端），自动攻击

### 一句话描述
> 你只有 5 分钟。移动闪避，自动攻击，每击杀一个敌人获得经验升级，从随机技能中选择强化自己的方式，在倒计时结束前击败 Boss 活下来。

---

## 2. 核心玩法循环

```
开始 → 选择角色 → 进入地牢
  ↓
移动闪避（唯一玩家操作）
  ↓
自动攻击最近敌人 → 击杀 → 掉落经验宝石
  ↓
拾取经验 → 升级 → 三选一技能（暂停）
  ↓
技能叠加 → 角色变强 → 敌人变多变强
  ↓
4:30 停刷怪 → Boss 出场 → Boss 战
  ↓
5:00 倒计时结束（或死亡/击败 Boss）
  ↓
结算界面（统计数据）→ 再来一局 / 返回菜单
```

---

## 3. 关键系统设计

### 3.1 移动系统
- 玩家只能控制移动方向（走位）
- 移动速度受技能影响（基础 120px/s）
- 碰撞墙壁时沿墙滑动

### 3.2 自动攻击系统
- 玩家自动向最近敌人发射投射物
- 攻击间隔受技能影响（基础 1.0s）
- 投射物类型受技能影响（默认：单发直线弹）
- 弹幕数量/速度/伤害均受技能影响

### 3.3 敌人系统
- 敌人从玩家视野外生成，向玩家移动
- 接触玩家造成伤害
- 击杀后掉落经验宝石（必掉）+ 随机掉落物

### 3.4 难度曲线
- 0:00-1:30 缓慢增加（生成间隔 4s → 2.5s）
- 1:30-3:00 中速增加（2.5s → 1.5s，新敌人类型出现）
- 3:00-4:30 快速增加（1.5s → 0.8s，高血量敌人增多）
- 4:30 停止普通刷怪，Boss 提示出现
- 5:00 倒计时结束

### 3.5 升级与技能系统
- 拾取经验宝石升级
- 升级时暂停游戏，从 3 个随机技能中选择 1 个
- 技能分类：
  - **数值型：** +20% 攻击力、+30% 移速、+15% 攻速
  - **弹幕型：** +1 弹丸、穿透弹、弹射弹、环形弹
  - **防御型：** 护盾（每10秒挡一次伤害）、击杀回血、受伤无敌时间延长
  - **特效型：** 冰冻光环（敌人减速）、燃烧弹（持续伤害）、闪电链
- 已获得的技能可以升级（Lv.1 → Lv.2），效果按公式增强
- 同类技能不重复出现

### 3.6 Boss 战
- 4:30 出现 Boss，有 2 秒出场动画
- Boss 拥有多阶段状态机：
  - Phase 1 (100%-50% HP)：基础攻击模式
  - Phase 2 (50%-20% HP)：新攻击模式 + 移速增加
  - Enraged (<20% HP)：全屏弹幕
- 击败 Boss 额外加分

## 5. 视觉风格

- **像素风**：精灵 16x16 / 32x32
- **配色方案**：深色地牢背景（#1a1a2e），暖色火光（#ff6b35），冷色魔法（#4ecdc4）
- **角色尺寸**：玩家 24x24，普通敌人 16-24x16-24，Boss 48x48
- **UI**：像素风 UI 框，位图字体
- **特效**：像素粒子、屏幕震动、浮动伤害数字

## 6. 音效设计

| 场景 | 音效 |
|------|------|
| 玩家射击 | 短促"嘟"声（800Hz, 0.05s） |
| 子弹命中 | 低沉"咚"（200Hz, 0.1s） |
| 敌人死亡 | 爆裂声 + 粒子 |
| 升级 | 上行和弦（C-E-G） |
| 拾取经验 | 清脆"叮"（1200Hz, 0.03s） |
| 受伤 | 低频冲击（100Hz, 0.15s） |
| Boss 出场 | 重低音轰鸣 + 屏幕震动 |
| 游戏结束 | 低沉下行旋律 |
