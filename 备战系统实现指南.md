## 4. 羁绊系统

### 羁绊激活机制

**SkillManager.js**：
```javascript
acquire(skill) {
  this.acquired.push({ ...skill });

  // 检测并激活羁绊
  const bond = getCharacterBond(this.characterId, skill.id);
  if (bond) {
    this.activeBonds.push({
      skillId: skill.id,
      bond: bond
    });
    console.log(`羁绊激活：${bond.desc}`);
  }
}
```

**羁绊效果应用**（getBuffs方法）：
```javascript
// 应用羁绊效果

      break;
    case 'slow_duration':
      buffs.bondSlowDurationBonus += bond.bonus;
      break;
    // ... 其他羁绊效果
  }
}
```

## 11. 技术细节

### 羁绊效果类型（bond.effect）

| effect类型 | buff字段 | 应用位置 |
|-----------|---------|---------|
| `range_up` | `rangeMult` | WeaponSystem（范围计算） |
| `slow_duration` | `bondSlowDurationBonus` | ProjectileSystem（冰锥创建） |
| `chain_jump` | `bondChainJumpBonus` | ProjectileSystem（闪电链） |
| `explode_radius` | `bondExplodeRadiusBonus` | ProjectileSystem（火球爆炸） |
| `orbit_blades` | `bondOrbitBladesBonus` | WeaponSystem（旋刃数量） |
| `damage_mult` | `damageMult` | 所有武器伤害计算 |

### 羁绊检测逻辑

```javascript
// SkillManager.acquire()
const bond = getCharacterBond(this.characterId, skill.id);

// characters.js
export function getCharacterBond(characterId, skillId) {
  const character = CHARACTERS[characterId];
  if (!character) return null;
  return character.bonds[skillId] || null;
}
```

**羁绊激活条件**：
- 角色ID匹配（characterId）
- 技能ID匹配（skillId）
- 技能被acquire（玩家选择该技能）
